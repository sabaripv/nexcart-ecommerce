NexCart – E-commerce Web App

Overview:

A modern premium looking e-commerce frontend built using React + Vite.
It focuses on clean UI/UX design.

Tech Stack:

Frontend: React, Vite
Styling: CSS / Tailwind 
Routing: React Router 
Build Tool: Vite

Features:

Responsive and premium e-commerce UI
Product listing layout
Category-based sections
Clean component structure
